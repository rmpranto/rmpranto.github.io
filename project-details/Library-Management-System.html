Library Management System

Comprehensive library and bookshop management application with dual modules for administrators and customers, built using C# and database technologies.

Role Undergraduate Developer
Tools
C# MySQL MSSQL Windows Forms
Timeline 2022
View on GitHub Back to Projects

Project Overview

During my undergraduate studies in a C# course, I developed a comprehensive Library Management System using C#, MySQL, and MSSQL, with Windows Forms for front-end development. This project enhanced my programming skills and provided practical experience in database management and user interface design, preparing me for future software development challenges.

Objectives

  • Create separate admin and customer modules with role-based access control
  • Implement complete CRUD operations for students, books, and loan records
  • Enable automated email notifications for loan reminders and updates
  • Generate invoices and maintain comprehensive transaction history

My Responsibilities

  • Designed and implemented database schemas in MySQL and MSSQL for data persistence
  • Developed Windows Forms interfaces for both admin and customer modules
  • Created backend logic for over 15 significant features including loan tracking, inventory management, and payment processing
  • Implemented email notification system for automated student communications
  • Built invoice generation and printing functionality for transactions

Key Features

Admin Module:

  • Manage student records and accounts with full CRUD capabilities
  • Add, update, and delete book entries with detailed metadata
  • Track and manage book loans with due date monitoring
  • Send automated email notifications to students regarding loans
  • View comprehensive history of all transactions and loans
  • Generate and print invoices for various transactions

Customer Module:

  • Search for books easily by title, author, or genre
  • Purchase books through a streamlined checkout system
  • Deposit money into customer accounts for future purchases
  • Receive order confirmation with automatically generated bills
  • Access transaction and loan history
  • Print invoices for completed transactions

Impact

  • Successfully implemented over 15 significant database-driven features enhancing system functionality
  • Provided hands-on experience with full-stack development using C#, database technologies, and UI design
  • Created a scalable system architecture applicable to various library and inventory management scenarios
