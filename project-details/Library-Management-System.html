Project Overview

Created a project on a Library Management System using MySQL, MSSQL, and C# with Windows Forms for front-end development.

Project Details

  • Initiated Admin, where admin can manage students, books, loan lists, send emails, show history, print invoices, etc.
  • Improvised book shop from where customers can Search for books, Buy books, Deposit money, and a bill will be generated after confirming the order.
