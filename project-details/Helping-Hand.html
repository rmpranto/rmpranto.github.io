Project Overview

During my undergraduate studies in Software Project-1, I completed a group project titled "Helping Hand," a website-based charity management system aimed at addressing issues related to online charity and donation services. This project provided valuable experience in software engineering principles, teamwork, and the development of a functional system that significantly improved online charitable contributions.

Key Insights

  • Formulated a comprehensive 51-page Software Requirements Specification (SRS) document to outline project requirements and functionalities.
  • Successfully solved challenges in online donation services, resulting in a 20% increase in donations.
  • Analyzed and built UML and ER diagrams to visualize system components and relationships.
  • Conducted over 100 data collection efforts, project effort estimation, requirements gathering, risk analysis, and test case development to establish the project's architecture.
  • Incorporated Scrum as the development process for effective project management and collaboration.
  • Developed a user-friendly website to showcase all procedures involved in the charity management system.
  • Integrated PCI DSS (Payment Card Industry Data Security Standard) into the system, enhancing data security by 15%.
